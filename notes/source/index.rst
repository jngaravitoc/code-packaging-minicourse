Python code packaging for scientific software
=============================================

By `Jo Bovy <http://astro.utoronto.ca/~bovy>`__

(last changed |gitTime|, rev. |gitHash|; last built |today|)

This is a brief set of notes with an overview of the steps involved in
developing, publishing, and maintaining a scientific software
package. The notes focuses on current best practices and the different
tools available to build, release, and maintain a scientific software
package.

A PDF version of these notes is available :pdf_link:`here <>`.
